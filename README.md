# Parlotte

A Matrix client that's fast, beautiful, and feels like it belongs on your device.

Parlotte is built from the ground up as a native experience on every platform — starting with macOS. No Electron, no web views. Just a snappy, polished messaging app powered by the open Matrix protocol.

## Why?

The Matrix ecosystem deserves a client that can stand next to the best commercial messaging apps. Parlotte aims to be that client — one that people actually *want* to use, not just one that technically works.

## Status

Early development. The Rust core is functional (login, rooms, messaging, sync) and tested against a real Matrix homeserver. The macOS app is next.
